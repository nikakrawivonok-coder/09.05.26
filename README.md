# Польовий Модуль — V4.1 Roles Fix

Ця версія виправляє ситуацію, коли посилання з `?role=gm` могло відкриватися як режим гравця через кеш браузера або старий service worker.

## Що змінилось

- файли стилів і скриптів перейменовано:
  - `styles.v4-1.css`
  - `app.v4-1.js`
  - `sw.v4-1.js`
- роль читається більш надійно з URL;
- підтримуються ролі:
  - `role=player`
  - `role=gm`
  - `role=master`
  - `role=майстер`
- локальне сховище тепер прив’язане до `room` і `player`.

## Посилання

Майстер:
`https://твій-нік.github.io/твій-репозиторій/?role=gm&room=zona01`

Гравець:
`https://твій-нік.github.io/твій-репозиторій/?role=player&room=zona01&player=fox`

## Якщо сайт все одно показує стару версію

Відкрий:
`https://твій-нік.github.io/твій-репозиторій/?role=gm&room=zona01&v=41`

Або очисти кеш сайту / відкрий у приватному режимі.
