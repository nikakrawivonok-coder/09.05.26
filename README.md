# Польовий Модуль — V5 Data Model

Це не повний онлайн-мультиплеєр, а правильний технічний фундамент для нього.

## Що додано

- структура даних кімнати `room`;
- підтримка кількох гравців у `players`;
- інвентар тепер належить конкретному гравцю;
- `Зараження` збережено як поле `infection`;
- вороги мають видиму частину і майбутню приховану GM-частину;
- журнал підтримує `visibility`;
- локальне збереження тепер прив’язане до кімнати;
- додано `syncAdapter`, який зараз працює локально, але підготовлений для Firebase/Supabase.

## Посилання

Майстер:
`?role=gm&room=zona01`

Гравець:
`?role=player&room=zona01&player=fox`

## Як оновити GitHub Pages

Заміни файли в корені репозиторію на:

- `index.html`
- `styles.v5.css`
- `app.v5.js`
- `manifest.webmanifest`
- `sw.v5.js`
- `README.md`
- папку `docs`

Після оновлення відкривай з параметром:
`?role=gm&room=zona01&v=5`
