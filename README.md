# Польовий Модуль — версія 3

Це версія, максимально наближена до другого референсного зображення:
- заголовок змінено на **Польовий Модуль**;
- замість **Радіація** використовується **Зараження**;
- підсилено схожість із референсом: темний метал, світлі паперові панелі, п’ять карток швидких дій, детальніший портретний блок, виразніша нижня навігація.

## Як оновити GitHub Pages
1. Розпакуй архів.
2. Заміні у своєму репозиторії GitHub старі файли на нові:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `manifest.webmanifest`
   - `sw.js`
   - `README.md`
3. Зроби Commit changes.
4. Зачекай 1–2 хвилини.
5. Якщо стара версія ще показується, онови сторінку без кешу або відкрий сайт у приватному режимі.
