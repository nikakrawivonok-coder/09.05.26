# Польовий Модуль — V4 Roles

Це крок 1 до гри з друзями: додано режими **Гравець** і **Майстер** через параметри посилання.

## Як запускати

### Для гравця
`?role=player&room=zona01&player=fox`

### Для майстра
`?role=gm&room=zona01`

Приклад:
`https://твій-нік.github.io/твій-репозиторій/?role=gm&room=zona01`

## Що змінилось

- з’явилась службова панель: кімната / роль / персонаж;
- у режимі `role=player` вкладка **Майстер** прихована;
- у режимі `role=gm` вкладка **Майстер** видима;
- підготовлено основу для наступного етапу: `room` і `player`;
- “КПК-попередження” перейменовано на “Сигнал модуля”;
- логіка додатка поки що локальна, без онлайн-синхронізації.

## Важливо

Це ще не мультиплеєр. `room` і `player` поки що тільки відображаються та готують код до підключення онлайн-бази. Наступний етап — підключити синхронізацію.
